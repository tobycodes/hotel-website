<h1>Simple Hotel Website</h1>

<p>This is a simple hotel website created using HTML and CSS.</p>

<p>The CSS preprocessor used for this project is Sass (SCSS).</p>

<p>The project also makes use of some neat and modern CSS techniques like Flexbox, BEM class naming system etc.</p>

<p>This project has also been designed to be fully responsive and supports all kinds of browsers.</p>
